PYTHON SCIPY RRT

Autonomous Drone Racing: Hierarchical Motion Planning

Algorithm Development & Optimization

Raw path planning algorithms like RRT often produce jagged, inefficient trajectories unflyable at high speeds. I developed a hierarchical framework to solve this.

Technical Implementation

  • Implemented 3D RRT to find initial collision-free topology through complex obstacle courses.
  • Formulated a Direct Transcription trajectory optimization problem, solving it using the SLSQP algorithm via SciPy.
  • Enforced physics-based constraints (terminal velocity), boundary conditions, and control jerk minimization.
  • Result: Reduced path length by 68.6% compared to raw RRT, generating smooth, flyable trajectories.
